# AndroidFileLog

#### 介绍
AndroidFileLog工具，可修改安卓应用。1、输出log至手机文件 2、屏蔽logcat查看log

#### 使用说明

![1](https://scimence.gitee.io/AndroidFileLog/files/pic/10.png)
